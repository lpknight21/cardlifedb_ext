About

This extension was designed to be used as a way to search for cards and build decks for Lord of the Rings: The Card Game.

Bug reports and suggestions can be sent to cardlifedb@gmail.com.

Future Features

Here are a list of features that are currently planned for future releases:

  • Deck Importing. This one is definitely coming soon!
  • Adding cost statistical graphs for every deck.
  • Campaign Tracker.
  • Deck Sharing Support.

Build Notes

Information about current and previous builds.

Build 1.0.1

  • The Grey Havens deluxe expansion has been added.

Build 1.0

  • CardLife DB is officially out of Beta!
  • OCTGN and Text file exports.
  • Accounts have been added to allow for syncing data across more than one machine.
  • Profile page for managing basic account information.
  • Sync Local decks to new account for those who had decks saved when app was still in Beta phase.

Build 0.3.3.4

  • OCTGN id's added for last two AP's in Angmar Awakens Cycle.

Build 0.3.3.3

  • Added cards for The Dread Realm.

Build 0.3.3.1

  • Fixed bug where cards with no traits were being taken out of search engine once you filtered by trait.

Build 0.3.3.0

  • Split out card search text boxes to search for name, trait and card text.
  • Cleaned up card data to allow for certain card names and traits to be searchable. Any card with a special character was unsearchable.
  • Added image sizing options for card search engine.
  • Added unique symbol to card popup text.
  • Updated BBCode deck text format to include the set for each card.
  • Fixed total deck card count for BBCode text format.
